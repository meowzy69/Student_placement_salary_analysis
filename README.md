# Student_placement_salary_analysis

## Project Overview

This Power BI dashboard analyzes student placement outcomes and salary trends to identify the key factors influencing employability and compensation. The project focuses on understanding how skills, internships, academic performance, and backlogs impact placement success and salary growth.

---

## Project Objective

The objective of this project is to analyze student placement and salary data and answer key business questions, including:

* What factors influence student placements?
* What drives higher salary packages?
* How do internships affect employability?
* Do backlogs reduce placement chances?
* Is CGPA a strong indicator of salary outcomes?

---

## Data Cleaning and Transformation

The following data preparation steps were performed:

* Renamed columns for consistency
* Corrected data types
* Replaced binary placement values with descriptive labels
* Removed inconsistencies in categorical fields
* Validated missing and duplicate records
* Prepared data for analysis and visualization

* Skill Score vs Average Salary

* CGPA vs Salary Analysis

---

## Key Insights

* Backlogs show the strongest negative impact on placement outcomes.
* Skill Score is the strongest predictor of salary growth.
* Internship experience improves both placement rates and salary outcomes.
* CGPA has a weaker influence on salary compared to skills and internships.
* Practical skills and industry exposure have a significant influence on employability.
